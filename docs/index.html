Khóa 34 & Khóa 03

📈 Phân tích Test 1 & Tỷ lệ Pass

Tương quan giữa điểm Test 1 và tỷ lệ hoàn thành khóa học. Phân loại học viên theo 4 nhóm tổng (All), <40, 40-50, 51-59, ≥60.

✅ Đã cập nhật

🎯 Giải thích cách chọn ngưỡng phân loại lớp

Phân tích chi tiết vì sao chọn ngưỡng 40% và 75% khi dùng %HV≥60 để phân loại lớp. Kèm biểu đồ trực quan hóa dữ liệu 89 lớp.

✅ Đã cập nhật

🔬 Backup số liệu: Tối ưu ngưỡng phân loại

Kiểm định thống kê toàn bộ 891 tổ hợp ngưỡng. Trả lời câu hỏi "Tại sao chọn 40% và 75% mà không phải con số khác?"

🆕 Mới

🎯 Đánh giá tính khả thi của Goal Pass Rate

Phân tích xem goal 80% (Nhóm A) / 70% (Nhóm B) / 60% (Nhóm C) có thực tế không. Chi tiết từng lớp đã đạt và còn xa.
